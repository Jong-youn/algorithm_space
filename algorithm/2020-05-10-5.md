## 문제  
회문 문자열 검사

## 풀이
```python
#1 리버스 
import sys
sys.stdin   = open("input1.txt", "rt")
n           = int(input())

for i in range(n): 
    word = ''.join(input()).lower()
    if word == word[::-1]:
        print('#%d YES' %(i + 1))
    else :
        print('#%d NO' %(i + 1))
```
