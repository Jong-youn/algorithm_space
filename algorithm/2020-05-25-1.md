## 문제  
ch.5-7
교육과정 설계  

## 풀이
```python
import sys
sys.stdin = open("input7.txt", "r")
comp = list(input())
n = int(input())
tt = []
for _ in range(n) :
    tt.append(input())

for word in range(n) :
    min = -1
    for i in comp :
        if i in tt[word] :
            if tt[word].index(i) > min :
                min = tt[word].index(i)
            else :
                print('#%d NO' % (word+1)) 
                break                
        else :
            print('#%d NO' % (word+1)) 
            break
    else :
        print('#%d YES' % (word + 1))
```
