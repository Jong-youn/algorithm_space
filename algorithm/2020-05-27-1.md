이진트리 순회(깊이우선탐색)  
전위순회, 중위순회, 후위순회  

## 풀이
```python
def aa(k) :
    if k > 7 :
        return
    
    else :
        aa(k*2)
        aa(k*2 + 1)
        print(k, end=" ")

aa(1)
```
